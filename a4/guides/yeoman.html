Guide - Setup Yeoman Workflow

Tools

Front-end web developers frequently use Yeoman for improving their productivity. It is comprised of three core tools:

  • A scaffolding tool, yo.
  • Package managers, bower and npm.
  • A build tool, grunt.

In addition, CSS preprocessors such as Sass are typically used. You will need to install compass for this.

Installation Guide

Environments

We need to install the following environments to run the framework. Guides for installation are provided in their websites:

  • Node.js and its package manager npm.
  • Ruby and its package manager gem.

We use npm to install yo, bower, and grunt:

sudo npm install -g yo bower grunt-cli

We use gem to install compass: 

gem update --system
gem install compass

Dependencies

We have already scaffolded the project structure with yo in the starter code.

Once you get the starter code, you will need to install dependencies with npm and bower. Dependencies are defined in package.json and bower.json, respectively. Issue the following commands at the project root will install the dependencies for you:

sudo npm install
bower install

You will see two directories being created: node_modules and bower_components. These are where the dependencies are stored.
